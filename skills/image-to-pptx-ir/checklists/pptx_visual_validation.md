# PPTX 高保真检查清单

## 1. 形状检查

- [ ] 外层大框直角/圆角是否与源图一致
- [ ] 内部卡片圆角是否一致
- [ ] 虚线框是否保留虚线
- [ ] 通道框是否为虚线
- [ ] UDP 框是否为圆角矩形
- [ ] 配置文件框是否为圆角矩形

## 2. 箭头检查

- [ ] 发送方向为实线
- [ ] 接收方向为虚线
- [ ] 箭头方向正确
- [ ] 箭头头部在正确端
- [ ] 一根箭头没有被拆成多段
- [ ] 黄色配置箭头为一侧一根完整虚线箭头
- [ ] 箭头没有被白色框遮挡

## 3. 文本检查

- [ ] 标题不换行
- [ ] 标签不换行
- [ ] 通道文字不换行
- [ ] 配置文件内容不换行
- [ ] 字体、字号、粗细接近源图
- [ ] PowerPoint 原生打开后不换行

## 4. 边界检查

- [ ] 所有子元素在父容器内
- [ ] 消息链表小块不越界
- [ ] 消息池小块不越界
- [ ] 箭头锚点不压边
- [ ] 左右镜像坐标已重新计算

## 5. 渲染检查

- [ ] PPTX 已导出 PNG
- [ ] 已与源图对比
- [ ] 已使用 PowerPoint 原生打开检查
- [ ] 所有修正已回写 render.json

## 6. 复杂图标素材检查

- [ ] 用户提供的透明素材或指定素材目录已优先使用
- [ ] 复杂图标没有被近似图标库资源替换
- [ ] 素材包含真实 Alpha 通道，棋盘格未烘焙进像素
- [ ] 发光、阴影和柔边保留半透明效果
- [ ] 在白色、黑色和页面背景上均无白边、灰边或颜色污染
- [ ] 已记录 sourceBBox 与 alphaBBox
- [ ] 回贴时使用 alphaBBox 对齐 sourceBBox，图标未拉伸或偏移
- [ ] 图标中不包含本应可编辑的文字、框、箭头和连接线
- [ ] 候选素材已合成回原位置并通过局部相似度复核
